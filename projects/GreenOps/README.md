# GreenOps 🌱

AI-Powered Cloud Sustainability & Optimization Platform

## Overview

GreenOps is an AI-assisted decision support system that analyzes cloud carbon footprint data and transforms it into actionable optimization insights.

The platform aims to:

- Analyze cloud emissions trends
- Detect inefficiencies and anomalies
- Recommend optimization actions
- Predict future carbon trends
- Introduce a Green Efficiency Score (GES)
- Gamify sustainability through badges and streaks

## Tech Stack

- Python
- FastAPI
- Google Cloud Platform
- BigQuery
- React
- Machine Learning
